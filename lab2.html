Lab 2 - Mobile Application Development Tool

    Tool Name: Android Studio
    Platform: android
    Application Development:
    • Type: Native & Web Apps
    • Advantages: Primary IDE for native Android App Development. Supports many languages.
    • Disadvantages: No Cross-Platform support

    Website: https://developer.android.com/studio/index.html
    Developers: Google, Jet Brains
    Release Date: December 2014
    License: Freeware +Source code
    Price: Free
    Programming Languages: Java, C++, Kotlin
    Features:
    • Flexible, Gradle-based build system
    • Fast and feature-rich emulator
    • Extensive tessting toolss & frameworks
    • C++ & NDK support
    • Official Android os IDE
    • Many more features

    Debugger: Android Studio has its own debugger. You can debug your code on specific devices and can set breakpoints in Java, Kotlin, and C/C++ code.
    Built-in Emulator: Contained in the IDE
    Host Machine Requirements:
    • Microsoft® Windows® 7/8/10 (32- or 64-bit)
    • Mac® OS X® 10.10 (Yosemite) or higher, up to 10.14 (macOS Mojave)
    • GNOME or KDE desktop
    • 4 GB RAM minimum; 8 GB RAM recommended.
    • 2 GB of available disk space minimum, 4 GB Recommended
    • Java Development Kit 8
    • 1280 x 800 Minimum screen resolution
    IDE: Android Studio has its own IDE
    Required Software for App Dev:None required
    Advantages:
    • Tons of features
    • Continued updates due to being an official IDE
    • Very large userbase, allowing more user support
    • Free to download and develop
    Disadvantages:
    • Requires a $25 registration fee to upload apps to the Google Play Store
    • Uses a lot of RAM during usage
    Languages: Contains a lot of languages, includinng the ability to use right-to-left scripts
    Licenses: Pay $25 registration fee to upload to Google Play Store
    3rd-Party Emulator:
    • Robotium Recorder: Robotium Recorder is an Android automatic testing tool. It allows a developer to automate test cases and user actions. The APK is easy to use and free to download.
    Market Share: 1.95%
    # Available Apps: 2.8 millionn
    # of Downloads: N/A
    First Announced: May 16, 2013
    Current Version: 4.0
